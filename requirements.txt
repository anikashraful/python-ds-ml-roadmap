To run the code in these notebooks, follow these steps:

1.  **Clone the repository:**
    ```bash

    cd python-ds-ml-roadmap
    ```

2.  **(Recommended) Create a virtual environment:**
    ```bash
    python -m venv ds-roadmap-env
    source ds-roadmap-env/bin/activate  # On Windows, use `ds-roadmap-env\Scripts\activate`
    ```

3.  **Install the required dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
